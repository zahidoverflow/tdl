üìñ How It Works

  1. Script auto-installs tdl.exe if not found
  2. Prompts for Telegram channel URL
  3. Downloads files in background
  4. Monitors disk usage (limit: 45GB)
  5. Uploads completed files to Google Drive
  6. Deletes local files after successful upload
  7. Repeats until entire channel is backed up
